Panel's Content from DOM Node

This demo demonstrates how to setup a SenchaTouch panel to be filled up with some content within the document's BODY.

  1. We need a DIV (a generic node) to be identified by an ID so we can reference to it from Ext
  2. We setup a panel using a Template who capture contents from an HTML node identified by ID
  3. We setup an empty data source to force Panel to render the Template.
